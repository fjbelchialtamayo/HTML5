El logo de HTML 5 no oficial se muestra más adelante.

HTML 5 presenta una serie de nuevas etiquetas y atributos orientados a resolver las necesidades del uso típico de los sitios web actuales. Algunos son técnicamente similares a las etiquetas div y span, pero tienen un significado semántico, como por ejemplo la etiqueta nav y footer. Otros elementos suministran nuevas funcionalidades a través de una interfaz estandarizada, como sucede con los elementos audio y video.

El logo que nunca llegó a ser.
